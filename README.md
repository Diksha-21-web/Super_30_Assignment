## 📌 Project Description
This is a console-based Student Management System developed using Core Java.  
The system allows users to manage student records using file handling (without using any database).

All student data is stored in a binary file using Java Serialization.

---

## 🎯 Objectives
- Implement File Handling (Read/Write)
- Apply Object-Oriented Programming concepts
- Use Serialization for storing objects
- Handle exceptions properly
- Validate user input

---

## ⚙️ Technologies Used
- Core Java
- File Handling (ObjectInputStream / ObjectOutputStream)
- Serializable Interface
- ArrayList
- Scanner
- Exception Handling
